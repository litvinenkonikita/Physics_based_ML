Файлы XXXX_XXXX.pt - 4-мерные 1x128x128x128 тензоры PyTorch/Numpy. В папках XXXX_XXXX хранятся поясняющие данные для этих тензоров, а именно:
1) 2-мерное изображение аорты с выделенной областью, которая попала в 3-мерный тензор.
2) папка modified_pngs, в которой 2-мерные 128x128 png-срезы 3-мерного тензора.
3) в папке reordered_pngs хранятся оригинальные 2-мерные срезы 3-мерного тензора (из датасета https://www.vascularmodel.com/dataset.html), 
которые были преобразованы в 128x128.
4) Для экспериментов предлагаю использовать эти тензоры: 0006_0001.pt, 0093_0001.pt и 0140_2001.pt. Граничные условия для тензоров
указаны в файле bc.txt в формате JSON:

{"0006_0001.pt": [[0.0, 0.0, -6.388888888888888517e-6], [0.0, 0.0, 6.388888888888888517e-6]], 
 "0093_0001.pt": [[0.0, 0.0, -2,777777777777777593e-7], [0.0, 0.0, 2,777777777777777593e-7]],
 "0140_2001.pt": [[0.0, 0.0, -5.555555555555554975e-6], [0.0, 0.0, 5.555555555555554975e-6]]}

, то есть чтобы их считать, можно написать такой код:

import json

with open('bc.txt') as f:
	data = f.read()

js = json.loads(data)

# Flow rates Q1-,Q2-,Q3-
Qm = js["0006_0001.pt"][0] # [0, 0, -6.388888888888888517e-6]
print(f'Qm : {Qm}')

# Flow rates Q1+,Q2+,Q3+
Qp = js["0006_0001.pt"][1] # [0, 0, 6.388888888888888517e-6]
print(f'Qp : {Qp}')
